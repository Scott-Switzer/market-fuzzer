ROBUSTNESS TEST

Test a trading strategy in new markets

What this does: we run your strategy in computer-generated markets it has not seen before. You learn where it works, where it breaks, and how confident we are. This is a research test—not a promise of profit.

Important: ACME and HEDGE were demo placeholders and are not real market data. The current demo tests synthetic instruments only. To use your own strategy, use the Strategy Stress Lab’s built-in policies or register an adapter there; this page is the sealed evaluation step after a strategy exists.
How it works: 1) connect a strategy · 2) choose a small test · 3) lock the submission · 4) run hidden markets · 5) review the report.

Step 1 · Connect your strategy

A strategy is the program that decides when to buy, sell, or do nothing. It must run in an isolated container so it cannot see the test secrets.

Advanced: add a container strategy

Step 2 · Choose the test

A world is one generated market. We test multiple families so your result is not tied to one simulator.

Step 3–5 · Run and read the result

Lock prevents changes after submission. Run puts the strategy in hidden markets. Reveal opens the evidence after the run is complete.
