# Limitations

- The current engine is a single-venue price-time-priority order-book prototype; it does not yet model cancellations, latency queues, or a full event-driven exchange kernel.
- No historical Level-2/Level-3 calibration has been performed.
- Agents use simple deterministic heuristics; their behaviors are not learned or calibrated to institutional order flow.
- Results are a software demonstration, not investment or execution advice.
